# nodejs-mongodb
is a Backend engine using NodeJs, Express and MongoDB.


## Installation
Make sure you system Installed `docker` `docker-compose` `nodejs` `make` `npm` `yarn`
```bash
git clone https://github.com/fajrulaulia/wispy-pond-1798
cd wispy-pond-1798
yarn
yarn start
```

## Usage

